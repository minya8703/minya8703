### 전문성과 가치 중심의 개발

9년간 백엔드 개발자로 일하며 금융, 항공, 제조, 유통 등 7개 이상의 산업 도메인에서 시스템을 구축했습니다. 일본 11년 거주 경험을 통해 글로벌 시각을 갖추었으며, 현재는 실무 경험을 쌓으면서 클라우드 네이티브 기술로 역량을 확장하고 있습니다.

**기술은 수단이며, 중요한 것은 사용자 문제 해결과 비즈니스 가치 창출**이라는 철학으로 개발합니다. 단순히 백엔드 개발에 한정하지 않고, 필요하다면 프론트엔드, 인프라, DevOps 등 서비스 전반을 이해하고 기여할 수 있는 풀스택 엔지니어를 지향합니다.
